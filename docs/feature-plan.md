## 2. 추가 기능 목록

### FR-09 난이도 선택
| 난이도 | 숫자 범위 | 최대 시도 |
|--------|---------|---------|
| 쉬움 | 1 ~ 50 | 15회 |
| 보통 | 1 ~ 100 | 10회 |
| 어려움 | 1 ~ 200 | 7회 |

### FR-10 최고기록 저장
- 난이도별 최소 시도 횟수 저장
- localStorage 활용 (브라우저 종료 후에도 유지)
- 최고기록 갱신 시 알림 표시

### FR-11 결과 화면
- 성공/실패 시 별도 오버레이 화면 표시
- 표시 내용: 정답, 시도 횟수, 최고기록 갱신 여부
- 다시하기 버튼 제공
