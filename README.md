# Number Guessing Game 🎮

> 바이브코딩 실습 프로젝트 — Claude AI와 함께 제작

## 프로젝트 개요
1~100 사이의 숫자를 맞추는 웹 기반 게임입니다.

## 게임 스펙 (2026-05-28 기획 확정)
- 숫자 범위: 1 ~ 100
- 최대 시도 횟수: 10회
- 힌트 방식: HIGHER / LOWER
- 플랫폼: 웹 (HTML/CSS/JS)
- 디자인 테마: 네온/다크 게임 느낌

## 기술 스택
- HTML5 / CSS3 / JavaScript (Vanilla)
- AI 바이브코딩 도구: Claude Sonnet (Anthropic)
